Reconstructor App Needed

You will need the Frappé Reconstructor App to launch the UI stored within this QR code.

We understand that our proposed framework currently requires a separate reconstructor application to be functional.

However, we envision our work as foundational building blocks for standardised protocols in the future which can allow QR codes to serve UIs, and these protocols can be pre-baked into OSs, akin to how modern smartphones have in-built QR code readers.
